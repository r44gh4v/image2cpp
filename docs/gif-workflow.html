GIF Workflow

image2cpp supports GIF timeline workflows so you can tune and export frame arrays for animation on constrained displays.

Recommended Workflow

  1. Upload GIF and verify resolution matches your display target.
  2. Use threshold/contrast tuning globally, then pause and tune specific frames as needed.
  3. Use a consistent naming strategy for frame arrays.
  4. Export in Arduino PROGMEM for memory-constrained devices.

Performance Notes

Large frame counts and high resolutions can increase processing time and firmware size. Keep dimensions compact for smooth playback on low-power MCUs.
